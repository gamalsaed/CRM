# 🚀 G-CRM Frontend

A modern, scalable CRM (Customer Relationship Management) frontend built with the latest technologies to deliver a fast, clean, and user-friendly experience.

---

## 🧠 Project Overview

This system helps small and medium-sized businesses manage:

- Leads
- Users & Roles
- Projects
- Notes & Activities

Instead of relying on Excel sheets, everything is centralized in a single, intuitive dashboard.

---

## ⚙️ Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Shadcn UI
- React Hook Form + Zod
- TanStack React Query
- NextAuth.js
- next-intl (i18n)

---

## 📦 Features

### 🔹 Leads Management

- View all leads
- Create / update / delete leads
- Add & remove notes
- Assign leads to users or projects
- Role-based UI (Admin / User / Data Entry)
- Analytics dashboard
- Excel import/export

### 🔹 Authentication

- Authentication handled using NextAuth.js
- Credential-based login (integrated with backend API)
- Session management & protected routes

### 🔹 Users & Roles

- Manage users
- Update user data
- Change roles
- Change password

### 🔹 Projects

- Create and manage projects
- Add/remove team members
- Assign team leaders
- Link leads to projects

---

## 🌍 Internationalization

- Multi-language support using next-intl
- Easy language switching
